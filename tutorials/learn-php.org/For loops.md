Tutorial
--------

For example, let's say we want to create a list of all the odd numbers between 1 and 10, and print
them out one by one.

    <?php
    $odd_numbers = [1,3,5,7,9];
    foreach ($odd_numbers as $odd_number) {
      echo $odd_number . "\n";
    }
    ?>
