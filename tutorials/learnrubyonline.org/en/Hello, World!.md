Tutorial
--------

To print things to the console, use "puts", like this
```
    puts 'I will be printed to the console!'
```
Everything between the '' will be printed to the console.


Exercise
--------
Print "Hello, World!" to the console using puts.

Tutorial Code
-------------
=begin
Fix This!
=end
puts 'Goodbye, World!'

Expected Output
---------------
Hello, World!

Solution
--------
=begin
Fix This!
=end
