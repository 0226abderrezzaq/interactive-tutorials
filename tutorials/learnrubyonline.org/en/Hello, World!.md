Tutorial
--------

To print things to the console, use "puts", like this

    puts 'I will be printed to the console!'

or,

    puts "I will be printed to the console!"
    
Everything between the `'` and `"` will be printed to the console.


Exercise
--------
Print "Hello, World!" to the console using puts.

Tutorial Code
-------------
    #Fix This!
    puts 'Goodbye, World!'

Expected Output
---------------
Hello, World!

Solution
--------
    #Fix This!
    puts 'Hello, World!'
